# Conception & Management de Bases de Données SQL 🗄️

## 📌 Présentation du Projet
Ce projet porte sur la modélisation complète d'une base de données relationnelle, de la phase de conception (MCD) à l'implémentation physique et l'extraction de données stratégiques.

## 🛠 Compétences Techniques
- **Modélisation :** Création de schémas relationnels optimisés (Normalisation 1NF, 2NF, 3NF).
- **Langage SQL :** - Définition de données (DDL) : `CREATE`, `ALTER`.
    - Manipulation de données (DML) : `INSERT`, `UPDATE`.
    - Requêtage complexe : `JOIN`, `GROUP BY`, `HAVING`, et sous-requêtes.
- **Optimisation :** Mise en place d'index pour accélérer la recherche d'informations.

## 📊 Cas d'Utilisation
 Gestion des stocks, suivi de production, ou gestion des commandes.

## 💻 Environnement
- **SGBD :** MySQL / Oracle Apex / Access. (choisis celui que tu as utilisé).
- **Outil de Modélisation :** MySQL Workbench
